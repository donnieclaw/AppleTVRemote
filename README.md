## 🌏 English
A powerful, high-performance Windows application to control your Apple TV. This version focuses on seamless bidirectional keyboard synchronization and precise remote interaction.

### 🌟 Key Features
- **Real-time Bidirectional Keyboard Sync**: What you see on Apple TV is what you see on your PC. No more input errors or character drops.
- **Physical Keyboard Mapping**: Use your PC's arrow keys, Enter (Select), Esc (Back), and Space (Play/Pause) to control the TV directly.
- **Context-Aware Enter Key**: In remote mode, Enter is "Select". In typing mode, Enter only submits text—no more duplicate characters.
- **Premium UI**: Modern, responsive design aligned with Apple's aesthetic.
- **Advanced Shortcut**: Press `Shift + T` to manually toggle the remote keyboard input.

### 🚀 Quick Start
1. **Download**: Download the entire repository contents (including `AppleTVRemote.exe` and `_internal` folder).
2. **Run**: Launch `AppleTVRemote.exe` from the unzipped folder.
3. **Connect**: Right-click the system tray icon -> "Open Remote" -> Settings -> Pair with your Apple TV via PIN code.

## 🇨🇳 中文
专为 Windows 用户打造的 Apple TV 遥放增强工具。该版本彻底解决了打字不同步、按键误触等痛点，提供如丝般顺滑的操作体验。

### 🌟 核心功能
- **全自动双向打字同步**：电视端弹出键盘，电脑端自动同步。支持快速输入，内置 300ms 防抖，彻底解决“123 变 1”等回退问题。
- **物理键盘完美映射**：支持 PC 键盘方向键控制、Enter（确认）、Esc/Backspace（返回）、Space（播放/暂停）。
- **情境感知回车键**：普通模式下是确定键；打字模式下仅同步内容，绝不导致字符重复。
- **精致 UI 设计**：统一 320px 宽度对齐，视觉比例和谐，PIN 码输入清晰醒目。
- **一键召唤打字框**：随时按下 `Shift + T`，手动开启/切换同步输入。

### 🚀 快速开始
1. **下载**：下载整个仓库中的 `AppleTVRemote.exe` 和 `_internal` 文件夹。
2. **运行**：双压缩包解压后，在文件夹内双击 `AppleTVRemote.exe` 即可启动。
3. **连接**：右键系统托盘图标 -> “打开遥控器” -> 设置 -> 扫描并输入 PIN 码配对。
